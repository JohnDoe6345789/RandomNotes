# Comprehensive Survey of 3D CAD Software Packages

## Parametric Solid Modellers

* **SolidWorks**: Mechanical design, feature-based parametrics, assemblies, drawings.
* **PTC Creo**: Enterprise-grade parametric and simulation suite.
* **Siemens NX**: High-end CAD/CAM/CAE with advanced surfacing and assemblies.
* **Autodesk Inventor**: Parametric modelling with strong interoperability.
* **FreeCAD**: Open-source parametric modeller with workbench architecture.
* **Onshape**: Cloud-native parametric CAD with real-time collaboration.
* **Alibre Design**: SMB-focused parametric modeller.
* **IronCAD**: Drag-and-drop design environment with parametric features.

## Direct Modellers (Explicit / Push-Pull)

* **SpaceClaim**: Direct modelling for rapid geometry changes.
* **Siemens Syncreon (Solid Edge Direct)**: Synchronous direct editing.
* **Shapr3D**: iPad-first CAD with direct and parametric blend.
* **Moment of Inspiration (MoI)**: NURBS-centric direct modeller.

## High-End AEC and Manufacturing Suites

* **Autodesk Fusion 360**: CAD/CAM/CAE hybrid with cloud-backed storage.
* **CATIA**: Aerospace and automotive master system for surfaces and assemblies.
* **Rhino + Grasshopper**: NURBS modeller with parametric graph-based design.
* **Autodesk Revit**: BIM platform for architectural modelling.
* **Archicad**: BIM modelling for full-building lifecycle.

## Mesh Modelling / Polygon Modelers (Not CAD, but used for CAD pipelines)

* **Blender**: Open-source mesh modeller and sculpting platform.
* **Modo**: Polygon modelling and procedural workflows.
* **3ds Max**: Mesh and animation system used in industrial viz.
* **Maya**: Animation and sculpting tool, occasionally used for concept design.

## Sculpting / Organic Modelling

* **ZBrush**: Industry-standard sculpting engine.
* **Mudbox**: Autodesk sculpting platform.
* **Nomad Sculpt**: Mobile-first sculpting.
* **3D Coat**: Sculpting, voxel modelling, PBR texturing.

## CAM-Integrated Solutions

* **Mastercam**: CNC toolpath design.
* **HyperMill**: Advanced 5-axis machining.
* **Fusion 360 CAM**: Integrated into Fusion.
* **SolidCAM**: Embedded CAM inside SolidWorks.

## CAE / Engineering Simulation Suites

* **ANSYS Discovery**: Real-time simulation with geometry editing.
* **SolidWorks Simulation**: FEA suite inside SolidWorks.
* **SimScale**: Cloud-based simulation.
* **COMSOL Multiphysics**: High-fidelity multiphysics modelling.

## Architectural Modelling / Construction CAD

* **SketchUp**: Lightweight architectural modelling.
* **Vectorworks**: CAD/BIM for architecture and entertainment.
* **Chief Architect**: Residential design tool.
* **Revit**: Fully parametric BIM.

## CAD for Electronics and Mechatronics

* **Autodesk Eagle + Fusion 360**: PCB + mechanical integration.
* **KiCad StepUp**: FreeCAD integration for PCB-to-CAD workflows.
* **Altium 3D PCB Integration**: Full mechanical digital twin.

## Open-Source and Hacker-Friendly CAD

* **FreeCAD**: Modular, parametric, Python-extensible.
* **OpenSCAD**: Code-based CSG modelling.
* **CadQuery**: Python API for parametric CAD.
* **SolveSpace**: Lightweight parametric modeller.
* **BRL-CAD**: Military-grade CSG modelling.

## Generative / Algorithmic CAD

* **Grasshopper (Rhino)**: Node-based design logic.
* **Houdini**: Procedural modelling used for engineering workflows.
* **nTopology**: Lattice, topology optimisation, advanced fields-driven modelling.

## Industrial Design Tools

* **Alias**: Automotive surfacing.
* **Rhino**: NURBS industrial design.
* **SolidWorks Visualize**: Rendering and visualization.
* **KeyShot**: Rendering often paired with CAD.

## CAD for 3D Printing and Maker Workflows

* **Fusion 360**: Hobbyists and pros.
* **FreeCAD**: Open-source pipeline.
* **Onshape Free Tier**: Cloud modelling.
* **PrusaSlicer/OrcaSlicer Integration Tools**: Not CAD but frequently paired.

## Cloud CAD Ecosystems

* **Onshape**: Full cloud-native mechanical CAD.
* **Vectary**: Browser-based modelling, mesh-forward.
* **Tinkercad**: Beginner-friendly block modelling.
* **SelfCAD**: Cloud mesh and simple CAD hybrid.

## Game and Simulation Modelling Tools (CAD-adjacent)

* **Unity ProBuilder**: Level geometry modelling.
* **Unreal Engine Modelling Mode**: Geometry editing.
* **Godot Mesh Tools**: Procedural geometry.

## CAD File Interoperability Notes

* Parametric kernels: Parasolid, ACIS.
* Neutral formats: STEP, IGES, JT.
* Mesh formats: STL, OBJ, PLY.
* Modern containers: 3MF, USD, USDZ.

## Emerging Trends

* Field-driven geometry.
* Implicit modelling (SDF-based).
* Cloud-synchronised parametrics.
* AI-assisted CAD feature recognition.
